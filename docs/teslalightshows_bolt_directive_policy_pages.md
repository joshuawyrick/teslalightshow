# TeslaLightShows.com Bolt.new Implementation Directive

## Objective

Update TeslaLightShows.com with clear trust, policy, footer, and compatibility language across the site.

The site must clearly state that TeslaLightShows.com is an independent digital service that creates custom light show files intended for compatible Tesla vehicles. The site must not imply affiliation, sponsorship, endorsement, approval, partnership, or official status with Tesla, Inc.

Do not display the owner’s personal name publicly. Use this public business wording throughout the website:

```text
TeslaLightShows.com is operated by a California-based sole proprietor.
```

If a DBA or registered business name is added later, replace that line with:

```text
TeslaLightShows.com is operated by [DBA OR BUSINESS NAME], a California sole proprietorship.
```

---

## Site-Wide Rules

Apply these rules to all pages, headings, buttons, upload screens, checkout screens, confirmation emails, download screens, and metadata.

1. Do not use Tesla’s official logo.
2. Do not make the site look like an official Tesla website.
3. Do not use language such as “official,” “approved by Tesla,” “Tesla approved,” “Tesla recommended,” “partnered with Tesla,” or “guaranteed to work on every Tesla.”
4. Replace “Tesla recommended specs” with:

```text
Designed for Tesla-compatible Light Show file structure.
```

or:

```text
Intended for compatible Tesla vehicles that support the Light Show feature.
```

5. Use this compatibility wording when describing the product:

```text
Custom light show files intended for compatible Tesla vehicles that support the Light Show feature.
```

6. Use this independence wording when describing the business:

```text
TeslaLightShows.com is an independent digital service and is not affiliated with, endorsed by, sponsored by, or approved by Tesla, Inc.
```

7. Use this trademark wording when referencing vehicle names:

```text
Tesla, Model 3, Model Y, Cybertruck, Model S, Model X, and related vehicle names are trademarks of Tesla, Inc. References to Tesla and Tesla vehicle names are used only to describe compatibility and intended file use.
```
